# DESIGN.md - Google Stitch Spec

## 1. Warna (OKLCH & Tailwind equivalent)
- Background: #FAF9F6 (Warm White / Ivory) - Menghindari putih steril.
- Text Utama: #1E293B (Slate 800) - Lembut untuk mata.
- Accent Utama: #7C3AED (Lavender / Violet 600) - Identitas premium lembut.
- Sukses/Diuntungkan: #10B981 (Emerald 500)
- Bahaya/Terhantam: #EF4444 (Red 500)
- Peringatan/Dilema: #F59E0B (Amber 500)

## 2. Tipografi
- Display Font: DM Sans (Bold/Black, Tracking-tight, untuk heading tegas tapi bersahabat).
- Body Font: DM Sans (Regular, Line-height 1.6 untuk keterbacaan tinggi).

## 3. Spacing & Elevation
- Base Spacing: 8px grid (4/8/12/16/24/32/48/64)
- Shadows: Sangat tipis (shadow-sm atau border tipis slate-100) untuk menghindari kesan kaku.

## 4. Do's and Don'ts
- **DO:** Gunakan ikon emoji yang relevan untuk memperjelas konteks (🚜, ⛵, 👷).
- **DO:** Gunakan warna background kontras lembut untuk memisahkan bagian.
- **DON'T:** Menggunakan tombol primary untuk semua aksi.
- **DON'T:** Menulis paragraf panjang tanpa jeda visual.
